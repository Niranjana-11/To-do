# 📝 To-Do List App

A simple and responsive To-Do List application built using modern frontend tools.  
This app helps users add, manage, and track daily tasks efficiently.


## 🚀 Features

- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- 📱 Responsive design  
- ⚡ Fast development setup using Vite  


## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- Vite  
- React


## How It Works
- Enter a task in the input field.
- Click the Add button to add it to the list.
- Mark tasks as completed when done.
- Delete tasks when no longer needed.
